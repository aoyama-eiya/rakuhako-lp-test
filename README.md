# ラクハコキット Landing Page

物流・梱包見積もりサービス「ラクハコキット」のランディングページです。

## 公開サイト
[https://aoyama-eiya.github.io/rakuhako-lp-test/](https://aoyama-eiya.github.io/rakuhako-lp-test/)

## 開発
Next.js + Tailwind CSS で構築されています。

### ローカルでの実行
```bash
npm run dev
```

### ビルドとデプロイ
Github Pages へデプロイされています。
```bash
npm run build
npm run deploy
```
